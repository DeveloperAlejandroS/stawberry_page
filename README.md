🍓 Dulce Tentación – Landing Page

Landing page desarrollada como prototipo para una marca de fresas con crema artesanales.
El objetivo del proyecto es presentar de forma clara el producto, destacar su propuesta de valor y facilitar el pedido directo por WhatsApp.

Este proyecto fue creado como ejercicio de conceptualización de una landing page para marketing digital.

📌 Características

Diseño tipo landing page

Secciones claras orientadas a conversión

Integración con WhatsApp para pedidos

Navegación suave entre secciones

Preparado para responsive design

Integración con Google Fonts

SEO básico y etiquetas Open Graph

🖥️ Secciones de la Landing

La página está compuesta por las siguientes secciones:

Hero

Presentación principal de la marca con llamada a la acción.

Propuesta de valor

Explica por qué el producto es diferente:

Fresas frescas

Crema artesanal

Opción vegana

Domicilio disponible

Menú

Productos principales ofrecidos.

Oferta del día

Sección promocional conectada con redes sociales.

Pedidos

Sistema simple de pedido con dos opciones:

Pedido a domicilio

Pedido para recoger

Ambos redirigen a WhatsApp con mensaje predefinido.

Contacto

Información de contacto y redes sociales.

📱 Integración con WhatsApp

Los botones de pedido utilizan enlaces directos al API de WhatsApp:

https://wa.me/<numero>?text=<mensaje>

Esto permite iniciar un pedido automáticamente desde el navegador o móvil.
